Humanova is a premium, accessible platform that connects **students**, **volunteers**, **charities**, **companies** and **universities**, with special focus on **students with disabilities**.

This repository contains the **frontend** only: React + Vite + TypeScript + Tailwind, with support for:

- Multi-role dashboards (6 roles)
- Light / Dark mode
- English / Arabic localization (i18next)
- Glassmorphism UI with Humanova colors (olive + cream + gold)
- Mock APIs + React Query
- (Optional) Three.js hero scene (currently disabled placeholder for stability)

---

## Tech stack

- **React 18** + **TypeScript**
- **Vite** (dev server & build)
- **TailwindCSS**
- **Framer Motion** (animations)
- **React Router DOM**
- **Axios** + **@tanstack/react-query**
- **i18next** + **react-i18next** (en / ar, RTL-aware)
- **Vitest** + Testing Library (unit tests)
- (Optional) **@react-three/fiber** + `three` for the 3D hero
